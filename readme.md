# calc.c - simple cli calculator

this is a simple command-line calculator written in c. it can perform basic math operations and provides a simple, interactive menu for users.

## features

- addition, subtraction, multiplication, division
- interactive menu with clear ascii box layout
- user can perform multiple calculations in a row
- checks for division by zero
- simple and clean command-line interface

## usage

1. compile the program using gcc:

```bash
gcc calc.c -o calc
```

2. run the program:

```bash
./calc
```

3. simply follow the prompts:
